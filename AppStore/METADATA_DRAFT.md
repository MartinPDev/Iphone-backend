# App Store metadata draft

## Name

Nexora

## Subtitle

AI-assisted trading control

## Primary category

Finance

## Description

Nexora brings your automated trading workspace to iPhone. Review bot status,
manage strategies, connect supported exchanges, and control your trading bots
from one focused dashboard.

Key features:

- Secure account access
- At-a-glance bot and strategy dashboard
- Strategy creation and management
- Supported exchange connections
- Trading-bot start and stop controls
- Testnet-friendly setup

Nexora stores its login token in the iOS Keychain. Exchange credentials are
sent to the configured Nexora backend when a connection is created and are not
stored by the iPhone app.

Trading involves risk. Nexora does not guarantee returns and is not financial
advice.
