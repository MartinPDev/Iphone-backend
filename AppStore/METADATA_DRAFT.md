## Description

Nexora brings your automated trading workspace to your phone. Review bot
status, manage strategies, connect supported exchanges, and control your
trading bots from one focused dashboard.

Key features:

- Secure account access
- At-a-glance bot and strategy dashboard
- Strategy creation and management
- Supported exchange connections
- Trading-bot start and stop controls
- Testnet-friendly setup

Nexora stores its login token in secure platform storage. Exchange credentials
are sent to the configured Nexora backend when a connection is created and are
not stored by the mobile app.

Trading involves risk. Nexora does not guarantee returns and is not financial
advice.
